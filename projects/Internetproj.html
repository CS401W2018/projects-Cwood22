Internet

The network that allows every computer online to connect with other computers, servers, and devices. It is a place where information is accessed and shared.

Web

The web stores the information that is accessed on the internet and runs on the internet. The web is what allows connected computers from the internet to share information.

How are the Internet and Web different?

The internet and web are different because the internet is the network on which computers connect to one another. The web runs on the internet that allows these computers to share information. The internet is the connection, the web is the sharing point of information.

ASCII and Unicode

ASCII and unicode are both standards of communicating and representing different types of text on a computer. Unicode is short for universal code which is a standard accross the entire world. ASCII is an American standard of code.

URL

URL is short for Uniform Resource Locator. It is a tool which is used for locating webpages on the web. It defines the address for a given html page.

Domain Name

A domain name is something that identifies a location on the interent. Domain names can be purchased and sold. They are a specific location on the internet.

HTTP

HTTP is called the Hypertext Transfer Protocol. It allows users to transmit media such as html. It allows user to make hypertext links to access data online.

HTTPS

HTTPS is the same Hyper Transfer Protocol which allows transmitting of data such as html and other files. HTTPS however has the added benefit of the "S" for security. HTTPS has security measures to encript and establish a secure connection to a server.

IP Address

An IP Address is a number that identifies a user of the internet. It uses location data and is assigned by internet providers. It can be spoofed or hidden with the use of VPN to further protect privacy.

DNS

DNS or Domain Name System is a system that translates domain names into ip address to locate a server.

W3C recommendations

W3C recommendations are a set of rules or guidelines that create a standard for how types of web systems work together. For example, w3c recommendations include standards for html and domains.

XML, XHTML, HTML, HTML5

XML, XHTML, HTML, HTML5, are all examples of coding languages used to house, show, and deliver data. XML is the most rigid of each of them and requires precision. XHTML is is a combination of XML and html that is an XML coded as an html file. HTML and HTML5 are just different and current versions of the html coding system.
