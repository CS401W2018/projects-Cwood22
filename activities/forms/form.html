Engagement Questions 11/13

What is JavaScript? How did it get its name? What are the two ways you can run JavaScript? What was the most popular front-end js library? What is the most popular front-end js library today?

  1. JavaScript is another programming language created by a guy form Sun Microsystems. Javascript is a programming language that runs inside the browser. The name was a marketing ploy to try to pull Java, which was really popular, into the web.
  2. The two ways you can run JavaScript are 1. In the DOM or browser, and 2. Node.js.
  3. The most popular front-end JavaScript library was Jquery.js.
  4. The most popular front-end JavaScript library today is React.js
