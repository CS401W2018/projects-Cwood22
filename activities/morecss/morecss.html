Answering engagment questions

  1. Block tags are most tags that create thier own margin padding and order. Inline tags are like <img> tags that stay in line. They fall inline with the tag they are next to or in.
  2. Margin is the space outside the box and pushes boxes from box tags away from others nearby. The border is what outlines a block tag. The padding is space between block tags that pads pixels between tags.
  3. In css you can use pixels, centimeters, percentages, inches, ems, rems and viewpoints, in this case, my css consists of pixel measurments.
  4. A figure tag would be used when you want to contain something like this image down below. You should use it when you want to seperate our content that is part of the main content of your page.
